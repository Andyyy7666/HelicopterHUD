# HelicopterHUD
Helicopter HUD that includes a few cool features.

# Discord
Join here for support or to report bugs
https://discord.gg/Z9Mxu72zZ6

# Features: 
* Slower takeoff 
* Slower landing
* Engine status
* Main rotor status
* Tail rotor status
* Altitude display
* Airspeed display
